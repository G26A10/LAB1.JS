## !!!Инструкции по запуску проекта!!!

1.Установите Visual Studio Code 
или другой текстовый редактор.

2.Установите Node.js
 для запуска JavaScript в локальной среде.

3.Откройте браузер (Chrome, Firefox, Edge) и вызовите инструменты разработчика (F12 → Console).

4.Создайте папку проекта и добавьте файлы: index.html и script.js.

5.Подключите script.js к index.html с помощью тега <script src="script.js"></script>.

6.Откройте index.html в браузере и следите за выводом в консоли и всплывающими окнами.

Работа с переменными и условиями
```javascript
let name = "Andrew";
let birthYear = 2006;
let isStudent = true;

console.log(name, birthYear, isStudent);

let score = prompt("Введите ваш балл:");
if (score >= 90) {
  console.log("Отлично!");
} else if (score >= 70) {
  console.log("Хорошо");
} else {
  console.log("Можно лучше!");
}

for (let i = 1; i <= 5; i++) {
  console.log(`Итерация: ${i}`);
}
```


Результат: переменные выводятся в консоль, пользовательский ввод баллов , цикл выводит номера итераций.

## !!!Ответы на контрольные вопросы!!!
++Присуствуют в документации++
1.Чем отличается var от let и const? 

1.Переменная var имеет функциональную область видимости видна как в функции, так и глобально 

2.Переменная let блочная область видимости означает что видна только в своем блоке можно изменить значение 

3.Переменная const блочная область видимости нельзя поменять нельзя поменять значение 

2.Что такое неявное преобразование типов в JavaScript? 

Неявное преобразование типов — это когда язык автоматически преобразовывает одного типа значение в другое без указания программиста 

3.Как работает оператор == в сравнении с ===? 

== Сравнивает значения после неявного преобразования типов. 

=== Сравнивает значения без преобразования типов.
